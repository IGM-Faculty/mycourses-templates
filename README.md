# Erika Mesh's custom myCourses schedule templates

## Warnings
- These are largely undocumented and set up to work ONLY with RIT's deployment of D2L.
- I update this if/when I need a new format or starting dates for a new semester.

## Basic instructions
1. Download the [zip file](esmesh_mycourses_templates.zip) in this repo
2. Use the [Manage Files](https://community.d2l.com/brightspace/kb/articles/3328-about-manage-files) tool to upload the zip file to your course shell and unzip it.
3. [Create a new content page using the template](https://community.d2l.com/brightspace/kb/articles/3381-use-templates-in-content)
   a. Use "Browse for a Template"
   b. Select "Course Offering Files"
   c. Choose the template folder & then select the template you want to use
